Basic Calculator

This calculator takes a mathematical expression and evaluates it while considering order of operations. Parenthesis are not supported, but addition, subtraction, multiplication, division, and exponents are.
For example, an input of "2^3*4-5" would yield 27.
The console can be examined after the calculation in order to see the creation of a binary tree to evaluate the expression.
